Parallel Computing on iOS devices

This project built a Travelling Salesman Problem solver App on iOS devices with the use of parallel computing. I constructed a protocol similar to MPI using iOS frameworks like the NSOperationQueue and Multipeer Connectivity to implement the protocol and the application. To balance the performance difference between different devices, I implemented dynamic load balancing which takes into consideration about devices' computing power and the amount of calculations remaining.

In terms of the result, the program achieved a speed-up of 1.877 compared with running the same amount of calculation on one single device. A brief result table is provided below. Details of the implementation and analysis can be found in the report which can be downloaded at the end of the page.
